# Go-Upload-Static-Files
Go Upload Static Files，使用 Go 实现静态文件上传服务

# 使用
```
默认上传到 uploads 目录，默认端口 8080

命令：
Go-Upload-Static-Files
Go-Upload-Static-Files -port=8081
Go-Upload-Static-Files --port=8081
```
